# 03_language_vs_bargain 图表说明

议题 3：德语 / 英语岗位要求水平与议价能力（`bargain_score`）的关系。  
议价可算样本：n=151（投标人数、平均报价与预算欧元列齐全，且预算上限 > 下限）。

`bargain_score` 定义简述：报价在预算区间中的相对位置 \(P\) 与投标人数对数 \(C\) 标准化后，按 \(0.5\cdot z(P)-0.5\cdot z(C)\) 合成（分数越高，相对议价越有利）。

## 图片

| 文件名 | 说明 |
|--------|------|
| `bargain_score_hist.png` | 议价分数整体分布直方图。展示全样本 `bargain_score` 的形态与离散程度。 |
| `de_level_vs_bargain.png` | 德语水平 × 议价箱线图。各 `de_level` 下 `bargain_score` 的分布对比。 |
| `en_level_vs_bargain.png` | 英语水平 × 议价箱线图。各 `en_level` 下 `bargain_score` 的分布对比。 |
| `de_vs_en_bargain_comparison.png` | 德英对比总图（左右并排箱线）：两侧为 de/en 水平与议价关系，标题标注各自 Spearman ρ，便于比较关联强弱。 |

## 数据表

| 文件名 | 说明 |
|--------|------|
| `bargain_scored.csv` | 议价明细表：语言水平、投标/预算、中间量 \(P,C,z_P,z_C\)、`bargain_score`、`bargain_tier` 等。 |
| `cross_de.csv` | 按 `de_level` 汇总：n、议价均值与中位数。 |
| `cross_en.csv` | 按 `en_level` 汇总：n、议价均值与中位数。 |
| `de_vs_en_comparison.csv` | 德英对比数值：各自相对 `bargain_score` 的 Spearman ρ 与 \|ρ\|。当前约：德语 ρ=0.23，英语 ρ=0.07。 |
