# 02_language_vs_score 图表说明

议题 2：德语 / 英语岗位要求水平与 Fair Pay 评分的关系。  
交叉样本：n=28（仅已打分的 hourly 岗位）。

语言编码：`de_level` 0–3（0=不需要 … 3=C1/C2）；`en_level` 1–3（1=入门 … 3=精通）。

## 图片

| 文件名 | 说明 |
|--------|------|
| `de_level_vs_pass_rate.png` | 德语水平 × Fair Pay 通过率柱状图。各 `de_level` 的通过率，柱上标注样本量 n。 |
| `en_level_vs_pass_rate.png` | 英语水平 × Fair Pay 通过率柱状图。各 `en_level` 的通过率，柱上标注样本量 n。 |
| `de_vs_en_score_comparison.png` | 德英对比总图（左右并排）：两侧分别为 de/en 的通过率柱状，标题标注各自 Spearman ρ，便于直接比较谁与得分关联更强。 |

## 数据表

| 文件名 | 说明 |
|--------|------|
| `language_all.csv` | 全样本（约 162 条）语言编码结果：`de_level` / `en_level` 及判定理由。 |
| `cross_de.csv` | 按 `de_level` 汇总：n、平均分、通过率。 |
| `cross_en.csv` | 按 `en_level` 汇总：n、平均分、通过率。 |
| `de_vs_en_comparison.csv` | 德英对比数值：各自相对 `fair_pay_score` 的 Spearman ρ 与 \|ρ\|。当前约：德语 ρ=0.50，英语 ρ=-0.33。 |
