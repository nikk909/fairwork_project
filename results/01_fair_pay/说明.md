# 01_fair_pay 图表说明

议题 1：可评定 Fair Pay 的 hourly 岗位整体情况与影响因素相关分析。  
样本：n=28（`type=hourly` 且预算欧元列齐全）；阈值 = 13.9 EUR/h。

## 图片

| 文件名 | 说明 |
|--------|------|
| `score_distribution.png` | Fair Pay 得分分布柱状图。横轴为 `fair_pay_score`（0/1），纵轴为岗位数；展示通过与未通过的数量对比。 |
| `hourly_avg_boxplot.png` | 按得分分组的平均时薪箱线图。对比 score=0 与 score=1 的 `hourly_avg_eur` 分布，红色虚线为阈值 13.9。 |
| `factor_correlation.png` | 影响因素与 `fair_pay_score` 的 Spearman ρ 横向条形图。正相关为蓝、负相关为红；用于比较各因子关联强弱。 |
| `top_factor_1_de_level_vs_score.png` | Top1 因子关系图：德语要求水平 `de_level` 各档的 Fair Pay 通过率（柱上标注 n）。ρ≈0.50。 |
| `top_factor_2_language_is_de_vs_score.png` | Top2 因子关系图：发帖语言是否为德语 `language_is_de` 与通过率。ρ≈0.47。 |
| `top_factor_3_en_level_vs_score.png` | Top3 因子关系图：英语要求水平 `en_level` 各档的通过率。ρ≈-0.33（负相关）。 |

## 数据表

| 文件名 | 说明 |
|--------|------|
| `fair_pay_hourly.csv` | 已打分 hourly 岗位明细：时薪三列、`fair_pay_score`、类别与链接等。 |
| `factor_correlation.csv` | 各影响因素对 `fair_pay_score` / `hourly_avg_eur` 的 Spearman 相关系数与样本量，按 \|ρ_score\| 降序。 |
