# 议价能力定义（项目级）

数据来源：`dataset/dataset.csv`。本数据集不含单个投标明细、成交价与雇主信息，议价能力只能在**项目级**用投标统计与预算区间构造代理指标。

所用字段：`bid_count`、`bid_avg`、`budget_min`、`budget_max`（可选控制：`type`）。

---

## 0. 样本清洗

去掉满足任一条件的样本：

- `bid_count` / `bid_avg` / `budget_min` / `budget_max` 缺失
- `budget_max <= budget_min`（分母为 0 或为负，无法计算区间位置）

说明：`bid_count`、`bid_avg` 约有少量缺失（约 8 条），分析时直接剔除即可。

---

## 1. 两个原始指标

**价格位置 \(P\)**（越高 → 工人侧议价越强）：平均报价在雇主预算区间 \([budget\_min, budget\_max]\) 内的相对位置。

\[
P_i = \frac{bid\_avg_i - budget\_min_i}{budget\_max_i - budget\_min_i}
\]

| 取值 | 含义 |
|------|------|
| `> 1` | 平均报价超出预算上限 |
| `0 ~ 1` | 落在预算区间内 |
| `< 0` | 低于预算下限 |

**竞争强度 \(C\)**（越高 → 工人侧议价越弱）：投标人数取对数，减轻极端大值影响。

\[
C_i = \log(1 + bid\_count_i)
\]

---

## 2. 标准化

\(P\) 与 \(C\) 量纲不同，合成前在**清洗后的全样本**上做 z-score：

\[
z(P_i) = \frac{P_i - \bar{P}}{s_P}, \quad
z(C_i) = \frac{C_i - \bar{C}}{s_C}
\]

其中 \(\bar{P}\)、\(s_P\) 为 \(P\) 的样本均值与标准差（\(C\) 同理）。

---

## 3. 合成议价能力分数

\[
Bargain_i = \alpha \cdot z(P_i) - (1 - \alpha) \cdot z(C_i)
\]

默认 **\(\alpha = 0.5\)**（等权），即：

\[
Bargain_i = 0.5 \cdot z(P_i) - 0.5 \cdot z(C_i)
\]

含义：报价越靠近（或超出）预算区间上方、投标人数越少 → \(Bargain\) 越高，工人议价能力越强。

---

## 4. 可选：离散为高 / 中 / 低

按 `bargain_score` 的分位数分组，便于与德语/英语水平做交叉分析：

- **低**：≤ 第一四分位数（Q1）
- **中**：Q1–Q3
- **高**：≥ 第三四分位数（Q3）

或按中位数切成高低两组。

---

## 5. 实现示意（Python）

```python
import numpy as np

df = df.dropna(subset=["bid_count", "bid_avg", "budget_min", "budget_max"])
df = df[df["budget_max"] > df["budget_min"]].copy()

df["P"] = (df["bid_avg"] - df["budget_min"]) / (df["budget_max"] - df["budget_min"])
df["C"] = np.log1p(df["bid_count"])

df["z_P"] = (df["P"] - df["P"].mean()) / df["P"].std(ddof=0)
df["z_C"] = (df["C"] - df["C"].mean()) / df["C"].std(ddof=0)

alpha = 0.5
df["bargain_score"] = alpha * df["z_P"] - (1 - alpha) * df["z_C"]
```

---

## 注意

- `fixed` 与 `hourly` 的 `bid_avg` / 预算含义不同（总价 vs 时薪）；算 \(P\) 时同一项目内比值仍可比，但若按绝对值或分类型比较，建议分组或分别标准化。
- 计算 \(P\) 前须先将 `budget_min` / `budget_max` / `bid_avg` 统一为欧元（与 `file/zh/数据处理.md` 第 1 步一致：`金额_EUR = 金额_原币 × rate_原币 / rate_EUR`）。
