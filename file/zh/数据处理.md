脚本将 dataset.csv 文件中的数据进行如下处理：

1. 将所有薪资列统一换算成欧元
2. 在 hourly 计价的岗位中，去掉缺失时薪字段的样本，计算最低时薪、平均时薪 ((min + max) / 2)、
   和最高时薪；根据**平均时薪**打分，得出一个得分列（0 或 1，即是否高于 13.9 欧元/小时）。
   画出分析图表，结合岗位对应的 job_category_names 或是其他的给出一些分析结论。
   保存结果到新的 csv / 图片，放到 results 对应议题文件夹。
3. 对德语水平进行一个定义：必须良好德语 C1/C2，流畅德语 B1/B2，基础德语 A1/A2，不需要德语；
   以及英语水平：A1/A2 入门，B1/B2 流畅，C1/C2 精通。对所有岗位分析，
   得到德语水平（0,1,2,3）、英语水平（1,2,3），以及英语/德语判断理由列。
4. 判断方式见下方「语言编码规则」。
5. 在 hourly 计价、且已 Fair Pay 打分的岗位中，研究英语/德语水平与岗位评分的关系。
6. 定义议价水平：采用 `file/zh/议价能力.md`（在议价可算的岗位中计算）。

### 1. 货币换算（所有金额列，全体样本）

- 公式：`金额_EUR = 金额_原币 × rate_原币 / rate_EUR`
  （`currency_exchange_rate` 相对 USD：先换 USD 再换 EUR）
- 需换算的列：所有金额列，至少包括 `budget_min`、`budget_max`、`bid_avg`
- Fair Pay 与议价分析均基于换算后的欧元金额（议价相对位置也基于已换算欧元后的同币种区间）

### 2. Fair Pay 打分（仅 hourly）

- **不做** fixed 岗位的 Fair Pay 打分
- 剔除规则：hourly 中 `budget_min` 或 `budget_max` **任一缺失**则剔除（分析时若发现其他必要字段缺失，同样剔除）
- 衍生：`hourly_min_eur`、`hourly_avg_eur = (min+max)/2`、`hourly_max_eur`
- **得分列仅 1 个**：基于**平均时薪**，`fair_pay_score = 1 if hourly_avg_eur >= 13.9 else 0`
- 导出：只导出有结果的列；结果文件自行命名，放入 `results/01_fair_pay/`
- 图表（按重要程度）：
  - **必要**：得分 0/1 分布；按 `job_category_names` 的通过率；时薪（avg）箱线图（可按得分或类别）
  - **可选有用**：min/avg/max 时薪分布对比；样本量较小类别可合并或标注
  - 太次要的不做
- 所有文字结论只针对本议题实际分析的样本

### 3–4. 语言水平编码（全体岗位）

编码：

- 德语 `de_level`：0 不需要 / 1 A1–A2 / 2 B1–B2 / 3 C1–C2
- 英语 `en_level`：1 A1–A2 / 2 B1–B2 / 3 C1–C2（**无 0**：默认至少需要英语入门）
- 理由列：`de_reason`、`en_reason`
  - **规则命中**（标签 / CEFR / 词汇库）：前缀 `[rule]`，理由须中英双语（`中文 / English`）
  - **规则推不出完整 de/en**（含「未发现德语要求」「默认英语入门」这类情形）：必须调用 AI（DeepSeek）辅助判定，前缀 **`[AI辅助]`**，理由须中英双语
  - **无 Key / API 失败**：前缀 `[fallback]`，理由须中英双语（如默认 de=0 / en=1）

「有标签」= `job_names` 中的 German/English 等技能标签 **或** `description`/`title` 中的显式 CEFR（A1–C2）等，两者都算。

**优先级（严格）：**

1. 标签 / 显式 CEFR / 规则词汇库映射 → 得到水平；若同一语言出现多个水平，**只取最低**
2. **不做**「无德语上下文→直接 de=0」「无英语线索→直接 en=1」的纯规则默认；这两类视为规则不完整，须走 AI 辅助
3. 若规则仍推不出**完整** de 与 en → 调用 DeepSeek（AI 辅助）填缺失项，再发帖语言兜底：
   - `language=en` → `en_level = max(en_level, 1)`，德语不变
   - `language=de` → `de_level = max(de_level, 1)`，英语不变
4. 口语化描述（如 “fluent German”）先走**自建英德词汇→等级映射表**；映射表覆盖不了的再交 AI

DeepSeek：按 `deepseek_ai_prompt.md`；规则推不出完整 de/en 时调用；理由前缀统一为 `[AI辅助]`。
